# Structs-FinalProject
Added the following operators to trivial: **, +=, -=, *=, **=, in, not in.

# Running tests
Running all tests
- in a bash terminal: bash run-tests.sh (note users may need to edit the run-tests.sh file to work with their version of Python)

Running individual tests
- in a bash terminal: (note users should change python3 to be the version of Python that they are using)
  - python3 tokenizer.py
  - python3 evaluator.py
  - python3 parser.py
  - python3 test_suite.t

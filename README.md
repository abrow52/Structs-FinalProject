# Structs-FinalProject
Added the following operators to trivial: **, +=, -=, *=, **=, in, not in.

# Running Application (bash terminal)
- python3 runner.py
- to exit type: quit

# Running tests (bash terminal)
Running all tests
- bash run-tests.sh

Running individual tests
- python3 tokenizer.py
- python3 evaluator.py
- python3 parser.py
- python3 test_suite.t

# Notes
- When compiling run-tests.sh users may need to edit the run-tests.sh file to work with their version of Python
- When compiling individual files users should change python3 to be the version of Python that they are using
